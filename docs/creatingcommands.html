Creating Commands

Quick tutorial on how to create commands.

WARNING!
Only do this if you have discord.js knowledge!

To add commands, navigate to the commands folder located in the src folder.
Open one of the category folders (or create a new one) and create a command file, e.g. ping.js.
Then, add the module exports (see example below).

Module Exports Example
module.exports = {
name: 'ping',
description: 'Ping command',
async execute(bot, message, args){
message.reply("Pong!");
}
};
