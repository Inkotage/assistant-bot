Installing

This page will guide you through setting up Assistant Bot V3

NOTICE
Do not install Assistant Bot V3 as root/sudo/admin for security reasons

Fedora / CentOS / RHEL

The latest version of NodeJS isn't available from DNF or YUM. You can run the following shell script to install the latest version.

Install NodeJS
curl -sL https://rpm.nodesource.com/setup_12.x | sudo bash -

One of our dependencies, Quick.DB, has a few extra requirements for install. Download those using DNF. Git is also needed as it is how we download the bot from GitHub in the next step.

Install Quick.DB Dependencies & Git
sudo dnf install -y make automake gcc gcc-c++ kernel-devel git

Now, you're all set to install our bot!

Download The Bot & Install NPM Dependencies
cd $HOME && git clone https://github.com/coolmlgplayer-js/assistant-bot && cd assistant-bot && npm i --save

Congrats, you're all done! To run the bot, run the command below!

Starting The Bot
./launcher.sh
