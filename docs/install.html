Installing

This page will guide you through setting up Assistant Bot V3

WARNING!
Do not install Assistant Bot V3 as a root or a sudo user for security reasons
REQUIREMENTS
  1. Node.js v12 or above
  2. A brain that can follow the installation steps

1. Run npm i in the command line

2. Rename the .env.example file to .env and put in your token and preferences

3. Run npm start in the command line

4. To update the bot, run node update.js in the command line
